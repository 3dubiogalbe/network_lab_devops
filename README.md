# network_lab_devops
Technology laboratory focused on network and cloud architecture in DevOps, along with data and web development


Esse é um projeto realizado pela universidade Anhembi Morumbi, que relaciona os conceitos de sistemas distribuídos e computação pararela, com um algoritimo desenvolvido em Python.

 // Projeto acadêmico de computação paralela e distribuída. //

Objetivo:
Implementar uma solução sequencial e paralela para comparar desempenho.

Tecnologias:
Python
Multiprocessing / MPI (se utilizado)
